To-Do App server API

RESTful JSON API

Retrieve all todos

GET /todos

Create a todo

POST /items

Body: object with title, description, due date

Retrieve a specific item

GET /items/{id}

Update a specific item

PATCH /todos/{id}

Attaching a JSON body with the possible below fields:

  • title (string)
  • description (string)
  • dueDate (string)
  • completed(boolean)

Delete a specific item

DELETE /todos/{id}
